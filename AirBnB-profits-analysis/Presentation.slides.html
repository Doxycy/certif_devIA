Analyse des données AirBnB pour une optimisation des avis clients¶

1. Importance des avis clients¶

Airbnb couvre actuellement plus de 100 000 villes et 220 pays dans le monde. Rien que dans la rgion de Madrid, AirBnB dispose de 17747 logements.

Contrairement une chambre d'hotel, les logements sur AirBnB ne suivent pas un standing impos. Les htes crent leurs propres annonces, et certains peuvent tre plus honntes que d'autres.

Dans ce contexte, les avis clients ont une double utilit :

  • Publicit : un logement avec une meilleure note sera plus attrayant
  • Vritable reprsentation du logement pour les clients

Essentiel que les logements au sein de AirBnB soient bien valus

2. Problématique :¶

  • Logements globalement trs bien valus (avec une moyenne de 92%)
  • Un certain nombre ont des notes trs faibles

Comment amliorer cette valuation moyenne en maintenant des revenus levs pour AirBnB ?

2. Solution : meilleur accompagnement des hôtes inexpérimentés¶

Pourquoi ?¶

  • Impossibilit d'carter simplement les logements les moins biens nots, ceci reprsenterait une perte de 277 686 dollars sur l'anne juste pour la rgion madrilenne.
  • Les logements les moins biens nots sont les logements les moins valus avec les hotes les moins expriments (dans une certaine mesure)

Comment ?¶

1- Systme de parainnage des "Inexpriments" par les "SuperHosts"

2- Services de "standing" hotelier : les chambres d'hotel prsentant les prix les plus levs (p-value<0.05) pour des valutations quasi-identiques aux autres types de chambres.
